Services

These are the services that we provide to our clients

Governance, Risk & Compliance

IS security strategy and roadmap development, security framework development , security posture review, risk assessment, gap assessment and compliance check, audits ( IT general controls and IS), compliance to regulatory bodies and contractual obligations, health check of IT systems and infrastructure.

Security Testing

The Goal of Security Testing is to identify threats in the system and to detect every possible security risk in the system. It includes Vulnerability Management , Penetration Testing , Secure Code Review, Risk Assessment, Posture Assessment, Web Application Security testing and Ethical Hacking.

Secure Network Architecture (SNA)

It involves segregation of networks into secure zones, placement of Security products in the network like placement of NAC , IDS/IPS, firewalls, Next generation Firewalls, WAF( Web Application Firewall), load balancer, APT , DDOS protection, Multi factor authentication for logins etc.

SOC Hosting and SIEM implementation

SOC hosting and it's product SIEM tool implementation and functioning.
